## 1. 产品概述
手写字模拟器是一款在线文档转写工具，用户可上传 Word、PDF、TXT 文档，系统将文档内容转换为手写字体效果，并支持多种手写字体、信纸背景、书写参数的自定义配置，最终可导出为图片或 PDF。

- 解决场景：需要手写作业、笔记、信件等场景，省去手写劳动
- 目标用户：学生、办公人员、需要手写风格内容的创作者

## 2. 核心功能

### 2.1 功能模块
1. **首页/工作台**：文件上传区域、设置面板、预览区域
2. **文件解析模块**：支持 docx、pdf、txt 三种格式的文本提取
3. **手写配置面板**：字体选择、信纸样式、字号、字间距、行间距、墨色调节
4. **实时预览模块**：Canvas 渲染手写效果，支持分页预览
5. **导出模块**：导出为 PNG 图片或 PDF 文档

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 工作台 | 顶部导航 | 品牌Logo、快捷操作、帮助入口 |
| 工作台 | 文件上传区 | 拖拽上传/点击上传，支持docx/pdf/txt，上传进度展示 |
| 工作台 | 文本编辑区 | 可对解析后的文本进行二次编辑 |
| 工作台 | 字体设置面板 | 手写字体选择（含预览缩略图）、字号、墨色 |
| 工作台 | 信纸设置面板 | 信纸样式选择（横线、方格、空白、牛皮纸等）、纸张颜色 |
| 工作台 | 排版设置面板 | 字间距、行间距、段落间距、页边距调节 |
| 工作台 | 手写预览区 | Canvas实时渲染、分页切换、缩放控制 |
| 工作台 | 导出操作栏 | 导出PNG、导出PDF、重置设置 |

## 3. 核心流程

用户进入工作台 → 上传文档或直接输入文本 → 系统解析并显示文本内容 → 用户调整手写配置（字体/信纸/排版）→ 实时预览手写效果 → 导出为PNG或PDF

```mermaid
    D --> E["选择手写字体"]
    E --> F["选择信纸样式"]
    F --> G["调整排版参数"]

    H --> I["导出PNG / PDF"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：暖米色 `#F5F0E6` 为主背景，墨棕色 `#3D2914` 为强调色
- **辅助色**：朱红印泥色 `#B5493B`、复古绿 `#5A7A5A`
- **按钮风格**：圆角复古质感，hover时有轻微阴影下沉效果
- **字体搭配**：标题使用衬线体（Cormorant Garamond），正文使用手写风格字体
- **布局风格**：左右分栏布局，左侧为设置面板，右侧为预览区，采用复古纸张质感背景
- **装饰元素**：纸纹纹理、装订孔效果、胶带装饰、复古边框

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI Elements |
|-----------|-------------|-------------|
| 工作台 | 文件上传区 | 复古虚线边框、纸张图标、拖拽hover效果、支持格式提示 |
| 工作台 | 设置面板 | 手风琴式折叠分组、卡片式选项、滑块控件、颜色选择器 |
| 工作台 | 预览区 | 纸张投影效果、分页器、缩放控制、Canvas纹理叠加 |
| 工作台 | 顶部导航 | 品牌Logo、操作按钮组、复古分割线 |

### 4.3 响应式
- 桌面端（≥1280px）：左右三栏布局（设置区 | 文本区 | 预览区）
- 平板端（768-1279px）：上下布局，设置面板折叠为顶部Tab
- 移动端（<768px）：单列布局，设置面板抽屉式展开，触控优化滑块

### 4.4 动效设计
- 页面加载：纸张飘落入场动画，元素淡入
- 文件上传：拖拽时边框发光+纸张抖动效果
- 配置变更：预览区平滑过渡（非全屏重绘）
- 导出按钮：点击时印章盖下动画
